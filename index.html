Motivation For Project

My motivation for creating this project was because I surprised to learn that Julia set fractals were created by iterating complex numbers through a very simple mathematical equation: f(z) = z^2 + c. It was surprisingly easy to create a user-interactive program that generates Julia sets from a user-provided startpoint and outputs an image.
